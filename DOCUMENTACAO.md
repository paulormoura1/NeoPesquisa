# NeoPesquisa

## Slogan

Aprenda a pesquisar. Descubra. Produza conhecimento.

## Objetivo

Criar o maior portal brasileiro de apoio à pesquisa científica.

## Público

- Graduação
- Pós-graduação
- Mestrado
- Doutorado
- Pesquisadores
- Professores

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Bootstrap
- GitHub Pages
